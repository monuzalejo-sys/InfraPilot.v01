# KN-009 — A pre-commit hook validates ORION memory on every commit of this repo: staged ch

**fact:** A pre-commit hook validates ORION memory on every commit of this repo: staged changes to memory/*/state.json or metrics.json trigger tools/validate-memory.mjs and an INVALID result blocks the commit. The versioned copy lives at tools/hooks/pre-commit; the ACTIVE copy at .git/hooks/pre-commit is NOT tracked by git, so after a fresh clone it must be reinstalled with: cp tools/hooks/pre-commit .git/hooks/pre-commit

**context:** Installed 2026-07-02 so schema enforcement doesn't depend on any agent remembering to run the validator. The validator also gained a --stats flag that prints a model-calibration table from metrics.json modelOutcomes.
