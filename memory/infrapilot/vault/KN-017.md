# KN-017 — The xlsx npm package (dependency of excel-export.ts and the prices/import flow) 

**fact:** The xlsx npm package (dependency of excel-export.ts and the prices/import flow) has high-severity vulnerabilities with no fix available via npm registry. The actual fix is to install the official tarball from cdn.sheetjs.com (version 0.20.3) instead of the npm-published version. This machine's agent permission classifier blocks agents from installing packages via arbitrary URLs, so this upgrade requires manual action by the user (npm install https://cdn.sheetjs.com/xlsx-0.20.3/xlsx-0.20.3.tgz or equivalent).

**context:** Found during this session's dependency/security pass; deferred as PEND-014 rather than force-installed.
