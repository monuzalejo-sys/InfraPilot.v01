# KN-019 — Claude Code built-in agent permission classifier enforces two hard blocks: (1) '

**fact:** Claude Code built-in agent permission classifier enforces two hard blocks: (1) 'npm install' from external URLs (tarball CDN) is forbidden — agents cannot bypass package registry lock-in (see PEND-014, xlsx SheetJS CDN); (2) INSERT/UPDATE/DELETE operations against shared databases (like infrapilot-app's Supabase project) are forbidden without explicit user approval — agents may SELECT/read, but must ask before persisting data to shared resources (see PEND-002 write test). Both are safety gates; require manual user action or explicit write permission outside the agent sandbox.

**context:** Discovered during this session (2026-07-05) when attempting to install fixed dependencies and test data writes. Complements KN-018 (agent spawn reliability + recovery protocol); these are the 'permission boundary' facts agents must respect.
